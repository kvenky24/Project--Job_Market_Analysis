# 📊 Job Market Analysis

## 📌 Project Overview

This project analyzes job market trends using Python, SQL, and Power BI. The objective is to identify hiring patterns, salary trends, in-demand skills, top companies, and popular job locations through Exploratory Data Analysis (EDA) and interactive dashboards.

The project demonstrates data cleaning, analysis, visualization, SQL querying, and business intelligence reporting.

---

## 📁 Dataset

- Cleaned_Jobs.csv

The dataset contains information such as:

- Job Titles
- Company Names
- Job Locations
- Salary Details
- Required Skills
- Experience Level
- Employment Type
